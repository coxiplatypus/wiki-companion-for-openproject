# Privacy Notice

## Data Collection

This extension does not collect or transmit telemetry.

## Local Data Storage

Data stored in browser extension local storage:

- Allowlisted origins (`allowedOrigins`)
- Global feature toggle (`features.wikiTocAutoCollapse`)
- Wiki TOC settings (`wikiToc`)
- Diagnostics toggle (`diagnostics`)
- Per-site feature overrides (`siteOverrides`)

## Data Sharing

No local configuration or state data is sent to external servers by this extension.

## Permissions

- `storage`: saves local extension settings.
- `tabs`: detects current tab origin in popup.
- `scripting`: registers content script for allowlisted origins.
- Optional host permissions: requested only for user-allowlisted origins.
